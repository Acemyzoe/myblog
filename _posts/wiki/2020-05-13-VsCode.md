# VsCode

- [Keyboard shortcuts for Linux](https://code.visualstudio.com/shortcuts/keyboard-shortcuts-linux.pdf)

- 行缩进: ctrl + [  、ctrl + ]

  > 选中按TAB右移，按SHIFT+TAB左移

- 整行上下移

  > alt+up/down

- **显示/隐藏左侧目录栏**: ctrl + b
- **控制台终端显示与隐藏**: ctrl + ~
- **关闭编辑器窗口** : ctrl + w
- **折叠代码**： ctrl + k + 0
- **展开代码**： ctrl + k + j
